Description of the algorithms used

1. Caesar

The caesar encryption is an ancient cryptography algorithm. It shifts each letter of the cipher text to the right a certain number of times, determined by the value of the key, which is an integer. To decrypt we simply need to shift each letter of the cipher text to the left the same number of times.

2. Vigenere

The vigenere encryption can be compared to the caesar encryption, but more complex. The key is multiple letters. It shifts each letter of the right by a certain number of times, determined by the position in the alphabet of the letter at the corresponding index in the key. For exemple, if we want to encrypt "vigenere" with the key "abc", we shift the letter 'v' by 1 time to the right, then the letter 'i' by 2 times, the letter 'g' by 3 times, the letter 'e' by 1 time, and so on. To decrypt we do the inverse process.

3. AES 128

The AES 128 encryption algorithm is divided in 10 rounds. It uses keys of 128 bits, 32 hexadecimal characters for this website The text is divided in blocks of 16 characters, and we apply to it 4 different operations :

  • Sub bytes
  • Shift rows
  • Mix columns
  • Xor round key

Each of these operations are reversible, so we can decrypt the cipher text with the key.
